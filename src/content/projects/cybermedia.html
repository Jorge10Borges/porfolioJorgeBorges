📘 CyberMedia

Plataforma Educativa para PDVSA

Descripción

CyberMedia fue una aplicación multimedia desarrollada alrededor del año 2000 para Petróleos de Venezuela (PDVSA), en un contexto donde el acceso a internet era extremadamente limitado —conexiones por módem, sin YouTube ni redes sociales.

La solución estaba compuesta por dos módulos independientes:

  • Editor de contenidos: herramienta para construir y sincronizar charlas o exposiciones con video, texto y materiales adjuntos.
  • Reproductor (Player): software destinado al usuario final para visualizar el contenido de manera estructurada, con navegación interactiva y soporte multimedia.

Tecnologías utilizadas

  • Lenguaje: Visual Basic 5.0 (posteriormente migrado a VB 6.0) [[7]](https://developer.mozilla.org/en-US/docs/Web/HTML)
  • Base de datos: Microsoft Access
  • Multimedia: Reproductor embebido
  • Distribución: CD-ROM ejecutable
  • Otros recursos: Interfaces personalizadas, optimización para ejecución desde medios físicos

Rol y contribuciones

Diseñé y desarrollé ambos ejecutables. En el módulo editor, implementé herramientas para importar y sincronizar los elementos multimedia...

Retos destacados

Entre los desafíos principales estuvieron: manejar eficientemente recursos multimedia desde CD-ROM, asegurar sincronización precisa entre texto y video, y desarrollar una arquitectura modular que permitiera a diferentes equipos crear y distribuir contenido educativo con facilidad. Todo esto dentro de las limitaciones técnicas propias de la época.

Impacto

CyberMedia fue implementado en diversas áreas formativas de PDVSA y se convirtió en una herramienta de referencia para la capacitación interna.
